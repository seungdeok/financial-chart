# financial-charts

Build financial-charts using typescript and google-charts

<br>

## Development Usage

```bash
$ npm install
$ npm run build
$ npm run start:stories
```

<br>

## Usage

```bash
$ yarn add financial-charts
$ #or
$ npm i financial-charts
```
